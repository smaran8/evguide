# Step 2 — Fix visual design language

Your site should not look like a standard template. It should use one unified design system.

## Use this style direction

### Color system

* Background: very dark charcoal or near-black
* Surface cards: dark grey with slight glass effect
* Accent: electric green or cyan
* Text: white with muted grey secondary text

### Typography

* Bigger headings
* Fewer words per block
* More whitespace
* Strong contrast between heading, subheading, and metadata

### Cards

All cards should use the same radius, spacing, and hierarchy. Right now the page mixes blog, news, car cards, and review cards without a single visual rhythm. ([EV Guide][1])

### Buttons

Use only 3 button styles:

* Primary solid
* Secondary outline
* Ghost text

No more scattered small action links.
